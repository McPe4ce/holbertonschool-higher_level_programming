# Holberton School – Higher Level Programming (Python)

Python projects and exercises completed as part of **Holberton School Higher Level Programming**.

The repository contains topic-based modules focused on writing clean, idiomatic Python and understanding higher-level concepts.

## Repository structure

- `python-serialization/` – Exercises related to serialization/deserialization and data interchange

## Getting started

### Prerequisites

- Python 3.x
- A Unix-like environment (Linux/macOS/WSL recommended)

### Clone

```bash
git clone https://github.com/McPe4ce/holbertonschool-higher_level_programming.git
cd holbertonschool-higher_level_programming
```

## How to use this repository

1. Enter a module directory.
2. Read the module `README.md` for requirements and task descriptions.
3. Run scripts with:

```bash
python3 <script_name>.py
```
